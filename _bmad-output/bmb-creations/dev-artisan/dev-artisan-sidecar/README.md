# dev-artisan-sidecar Sidecar

This folder stores persistent memory for the **Master Artisan** Expert agent.

## Purpose
Le Dev-Artisan doit conserver une mémoire de ses patterns de refactoring et de l'historique de la dette technique pour assurer une harmonisation durable.

## Files
- memories.md: Standards de code, snippets favoris et contexte de réalisation des stories.
- instructions.md: Protocoles de démarrage et principes de craftsmanship.

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/dev-artisan-sidecar/{filename}.md`
