# pm-architecte-sidecar Sidecar

This folder stores persistent memory for the **Visionary Architect** Expert agent.

## Purpose
Le PM-Architecte doit maintenir une vision à jour des règles métier et de la structure globale du projet.

## Files
- memories.md: Vision du projet, règles métier centrales et décisions architecturales.
- instructions.md: Protocoles de démarrage et principes de garde-fous.

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/pm-architecte-sidecar/{filename}.md`
