# analyste-mentor-sidecar Sidecar

This folder stores persistent memory for the **Detective Mentor** Expert agent.

## Purpose
L'Analyste-Mentor doit charger sa mémoire (sidecar) à chaque démarrage pour rester utile sur la durée du projet.

## Files
- memories.md: Profil utilisateur, contexte projet et historique des sessions.
- instructions.md: Protocoles de démarrage et garde-fous pédagogiques.

## Runtime Access
After BMAD installation, this folder will be accessible at:
`{project-root}/_bmad/_memory/analyste-mentor-sidecar/{filename}.md`
